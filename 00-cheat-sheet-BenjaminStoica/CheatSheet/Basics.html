HTML

There are three components in a html element:

  • Start Tag
  • Element Content
  • End Tag

In HTML, we refer to elements without content as void elements (or empty elements).
These elements are identified by their self-closing syntax, eliminating the need for an end tag.
Here are some common examples for void elements in HTML:

  • br: (Line Break)
  • img: (Image)
  • a: (Link)
  • And many more like
    • base
    • col
    • command
    • input
    • meta

HTML attributes supply extra details about HTML elements, offering the means to adjust their
behavior, appearance, or furnish metadata. Here's how attributes are indicated in HTML:

  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"
  • Some examples for attributes: href, class, id

CSS

There are three components in a css element:

  • Selector
  • Property Name
  • Property Declaration

CSS properties use "combinators" to refer to explicit elements

  • ">" for direct children for example: article > h1
  • " " for descendants of any type
  • "+" for sibling (elements located DIRECTLY after another)
  • "~" for elements that come after another (Can have others between them)
