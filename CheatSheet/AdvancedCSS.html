CSS Classes and IDs

There are two ways to address elements

  • Inline css (which we want to avoid)
  • CSS classes

There are two types of attributes to address elements

  • Classes
  • Ids

Classes

Classes are defined with "<... class="class-name">content<...>

Are used to make an element addressable from an internal / external CSS

Are used if html element appears more often than once in the page

Examples for Classes:

  • table.personal_data
  • p.message
  • Or just: .special

IDs

Classes are defined with "<... id="id-name">content<...>

Are used to make an element addressable from an internal / external CSS

Are used if html element appears only once in the page

Examples for Ids:

  • body#mainpage
  • article#teaser
  • Or just: #special

CSS Box Model

Block elements take an extra line

  • article
  • section
  • p
  • h1(1-6)

inline elements are in the "normal" text flow

  • img
  • strong
  • span
